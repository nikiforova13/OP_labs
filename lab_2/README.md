# Лабораторная работа 7. Вариант 8

## Задание

Разработать функцию, которая удаляет из строки заданное количество символов, начиная с заданной позиции.

## Требования

- Строки представляются как массив символов (`char[]`)
- Запрещено использовать функции из библиотек `string.h` и `ctype.h`
- Запрещено использовать класс `string` из STL
- Реализована проверка на утечки памяти через `_CrtSetDbgFlag`
- Все утечки памяти устранены

## Реализованные функции

### `stringLength(const char* str)`
Вычисляет длину строки без использования `strlen`.

**Входные данные:**
- `str` – указатель на строку

**Выходные данные:**
- Длина строки (количество символов до `\0`)

### `removeCharacters(const char* str, int position, int count)`
Удаляет из строки заданное количество символов, начиная с заданной позиции.

**Входные данные:**
- `str` – указатель на исходную строку
- `position` – позиция начала удаления (индексация с 0)
- `count` – количество символов для удаления

**Выходные данные:**
- Указатель на новую строку с удалёнными символами (динамически выделенная память)
- `nullptr` в случае ошибки (некорректные параметры)

**Особенности:**
- Если `position + count` превышает длину строки, удаляются все символы от позиции до конца
- Выделяет память для новой строки
- Память должна быть освобождена вызывающим кодом через `delete[]`

## Структура программы

- `setConsoleEncoding()` – установка кодировки для русского текста
- `stringLength()` – вычисление длины строки
- `stringCopy()` – копирование строки (вспомогательная функция)
- `removeCharacters()` – основная функция удаления символов
- `printString()` – вывод строки на экран
- `main()` – основная функция с тестированием

## Примечания

- Программа использует динамическое выделение памяти для результата
- Все выделения памяти освобождаются корректно
- Проверка утечек памяти выполняется автоматически при завершении программы
